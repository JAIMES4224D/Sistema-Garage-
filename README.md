# 🚗 SistemaGarage

SistemaGarage es una aplicación de escritorio desarrollada en C# que permite gestionar el ingreso y salida de vehículos en un garaje o estacionamiento.  
El sistema registra las placas de los vehículos, controla la hora de entrada y salida, calcula el costo del servicio y permite visualizar el historial de operaciones.

---

## 📌 Características

- Registro de ingreso de vehículos
- Registro de salida de vehículos
- Cálculo automático del costo del estacionamiento
- Visualización del historial de vehículos
- Exportación de datos a Excel
- Visualización de autos actualmente en el garaje
- Resumen del total recaudado por día

---

## 🖥️ Interfaz del Sistema

El sistema cuenta con una interfaz sencilla donde el usuario puede:

- Ingresar la **placa del vehículo**
- Registrar la **entrada del vehículo**
- Registrar la **salida del vehículo**
- Calcular el **costo total del servicio**
- Visualizar los registros en una tabla

---

## 📊 Funcionalidades Principales

### Registrar Entrada
Permite registrar la placa del vehículo y guardar automáticamente la hora de ingreso.

### Registrar Salida
Calcula el tiempo que el vehículo estuvo en el garaje y determina el costo del servicio.

### Historial de Vehículos
Permite visualizar todos los registros realizados durante el día.

### Exportar a Excel
Los datos registrados pueden exportarse para su análisis o almacenamiento externo.
