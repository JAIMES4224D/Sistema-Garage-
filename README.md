# Garage Management System

## Description
A comprehensive garage management system that allows users to manage vehicles, services, and transactions efficiently.

## Features
- Vehicle Registration
- Service Scheduling
- Invoice Generation
- User Management
- Reports and Analytics

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JAIMES4224D/Sistema-Garage-.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Sistema-Garage-
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
- Start the application:
  ```bash
  npm start
  ```
- Access the application through `http://localhost:3000`.
